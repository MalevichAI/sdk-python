# Функции

Функции являются основными строительными блоками в Malevich SDK. Они позволяют определить отдельные операции, которые будут выполняться в вашем конвейере.

## Что такое функция?

Функция в Malevich SDK представляет собой обёрнутую операцию, которая может быть выполнена как часть большого рабочего процесса.

## Основные концепции

- Декларативное определение функций
- Типизация аргументов
- Композиция функций

Изучите подразделы для получения более подробной информации о работе с функциями.
