# Композиция функций

Изучите, как объединять несколько функций для создания сложных рабочих процессов.

## Цепочка функций

Функции могут быть объединены в цепочки для создания конвейеров обработки данных.

```python
result = function1() >> function2() >> function3()
```

## Параллельное выполнение

Функции также могут выполняться параллельно для повышения производительности.
